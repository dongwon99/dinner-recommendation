# 저녁 메뉴 추천

저메추 시스템은 사용자의 선호도를 기반하여 음식 DB로부터 음식을 가져와 추천해주는 시스템입니다. 음식의 종류, 맛, 조리 방식을 선택가능하고 해당되는 음식들의 이름과 종류, 가격이 출력됩니다.
